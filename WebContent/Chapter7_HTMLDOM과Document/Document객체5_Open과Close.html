새로운 document객체 열기와 닫기

		현재의 브라우저 윈도우에 HTML텍스트를 모두 지우고 새로운 내용을 출력하는 방법
	
		1) document.open() 
			- 기존 내용을 모두 지우고 document객체를 새롭게 열고.
			- 새로운 HTML 텍스트를 받아들일 준비를 한다.
		2) document.write()
			- 새로운 HTML 텍스트를 document객체에 담는다.
		3) document.close()
			- HTML텍스트를 모두 쓴 후에는 당연히 닫기를 해 놓아야 한다.
		cf) window와 document
			- document는 HTML문서 전체를 나타내는 객체이다.
			- window는 document 객체가 그려지는 브라우저 윈도우를 나타내는 객체이다.
			그러므로 window가 닫히면 (소멸되면) 그속에 있던 document 객체는 당연히 소멸된다.
